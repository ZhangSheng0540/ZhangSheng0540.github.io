hexo发生error：spawn failed错误的解决方法

发表于 2021-05-12 更新于 2021-06-04 阅读次数： Valine：

问题描述

  • 执行hexo d 命令的时候出现这个错误

造成原因

  • 因为git进行push或者hexo d的时候改变了一些.deploy_git文件下的内容。

解决办法

  • 删除.deploy_git文件夹;
  • 输入git config –global core.autocrlf false
  • 然后，依次执行：

    hexo clean
    hexo g
    hexo d
